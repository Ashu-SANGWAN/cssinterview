Q 1. What is box sizing?

The CSS box-sizing property allows us to include the padding and border in an element's total width and height.

Q 2. What is animation delay?

The animation-delay property specifies a delay for the start of an animation. The animation-delay value is defined in seconds (s) or milliseconds (ms).

Q 4. What is grid-template-columns used for?

The grid-template-columns property specifies the number (and the widths) of columns in a grid layout. The values are a space separated list, where each value specifies the size of the respective column.

Q 5. Which property you will use to merge cells vertically in grid?

Q 6. Which property you will use to merge cells horizontally in grid?

CSS grid-row property. is used to merge cells horizontally

grid-row-start and grid-row-end
